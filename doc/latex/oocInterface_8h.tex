\doxysubsection{Detailed Description}
Interface support for the OOC (Object-\/\+Oriented C) framework. 

This header defines the basic interface infrastructure for the OOC framework. Interfaces provide a way to define contracts that classes can implement, similar to interfaces in object-\/oriented languages like Java.
