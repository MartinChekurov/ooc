\doxysubsection{Detailed Description}
Class modifier definitions for the OOC (Object-\/\+Oriented C) framework. 

This header defines modifier flags that can be applied to classes in the OOC framework, similar to access modifiers and class modifiers in object-\/oriented languages like Java.
