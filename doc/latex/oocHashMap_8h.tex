\doxysubsection{Detailed Description}
Hash\+Map implementation for the OOC (Object-\/\+Oriented C) framework. 

This header defines the Hash\+Map class that implements the Map interface using a hash table with separate chaining. It makes no guarantees about the iteration order of entries. This implementation permits null keys and null values. Performance\+: O(1) average case for get, put, and remove operations.
