\doxysubsection{Detailed Description}
Abstract base class for List implementations in the OOC framework. 

Similar to java.\+util.\+Abstract\+List, this class provides skeletal List behavior on top of random-\/access primitives. Subclasses should implement at least size() and get\+At(index). For modifiable lists they should additionally override set\+At(), insert\+At(), and remove\+At().
