Detailed Description

Abstract base class for List implementations in the OOC framework.

Similar to java.util.AbstractList, this class provides skeletal List behavior on top of random-access primitives. Subclasses should implement at least size() and getAt(index). For modifiable lists they should additionally override setAt(), insertAt(), and removeAt().
