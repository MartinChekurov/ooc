Detailed Description

LinkedHashMap implementation for the OOC (Object-Oriented C) framework.

This header defines the LinkedHashMap class that implements the Map interface using a hash table with separate chaining and a doubly-linked list running through all entries to maintain insertion order. This implementation permits null keys and null values. Performance: O(1) average case for get, put, and remove operations. Iteration order: Entries are returned in the order they were inserted.
